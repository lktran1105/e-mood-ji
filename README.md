# E-Mood-ji ✨

Turn your emojis into something magical — albums, bedtime stories, and constellations.

## What it does

Pick 3–5 emojis and choose a mode:

- **Album** — suggests a real existing album matching your emoji vibes
- **Bedtime Story** — generates a whimsical short story woven around your emojis
- **Constellation** — maps your emojis as stars in a real constellation

Powered by Google Gemini API.

## Setup

1. Get a [Google Gemini API key](https://aistudio.google.com/apikey)
2. Open `index.html` in your browser
3. Paste your API key when prompted (saved locally in your browser)
4. Pick emojis and generate!

## Tech Stack

- Vanilla HTML/CSS/JS (single file, no build step)
- Google Gemini 2.0 Flash API
- Space Grotesk + DM Sans fonts
